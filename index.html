Protocol
›
Noise Protocol Framework
WireGuard uses Noise_IKpsk2. Initiator knows responder's static key. PSK for post-quantum resistance. 1-RTT handshake.
›
ChaCha20-Poly1305
Stream cipher + MAC. Faster than AES on ARM (no hardware AES on Pi). Constant-time implementation. No timing attacks.
›
Curve25519
Elliptic curve Diffie-Hellman. 128-bit security. 32-byte keys. Fast on ARM. Used for key exchange and identity.
›
Key Rotation
Keys rotate every 2 minutes (rekey after 2^60 messages or 120 seconds). Forward secrecy — compromise one key, old traffic stays safe.
Performance
›
Throughput on Pi 5
~400 Mbps WireGuard throughput on Pi 5 (kernel module). Limited by Ethernet, not crypto. Effectively line-rate for 1Gbps.
›
Latency Overhead
~0.5ms added per hop. Negligible for most workloads. Sub-millisecond handshake. Stateless — no connection setup cost.
›
Battery/Power
Minimal power impact. ChaCha20 is efficient on ARM. No always-on background process — kernel module activates on traffic.
